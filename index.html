  • Sublime Text

    I've made some packages for the Sublime Text editor

    Sublime Gulp

    Sublime Gulp


    A Gulp task runner for Sublime Text. Pick a task from all available gulpfiles and run it directly from Sublime, and see the result highlighting.

    This package also provides some gulp snippets, a listing for all gulp plugins and some other useful settings for running the tasks.


    Sublime Jasmine

    Sublime Jasmine


    A utility package to work with Jasmine, the Javascript BDD framework.

    After installing you'll have: Jasmine syntax highlighting for your spec files, snippets for the most common functions and commands for creating and switching between code and spec files.

  • Langy

    Langy is a tool that simplifies input source languages on OSX.
    Forget about changing the language from Spanish chat to Russian markdown to your Greek studies.

    Set up a default language, a language for specific apps and on each switch it'll be ready to type.
